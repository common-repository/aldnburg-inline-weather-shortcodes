Display weather information via shortcode. No fancy widget, just plain text. Current or forecast.

== Description ==
This plugin allows you to display weather information in your written text. The current data and the forecasts are pulled from openweathermap.org. 
There are no widgets and no disturbing colours, just plain information. Use it on your travel blog, your tourist information, sport event.... wherever you want!

= Features (so far): =

* [temp] shortcode for current temperature.
* [tempDay] shortcode for a forecast (up to five days).
* Decide between metric or imperial units.
* Display unit symbols automatically.

You need an API-Key from [openweathermap.org](http://openweathermap.org/). Follow the link and sign up for the plan you need.

= Languages: =

The plugin is fully translatable. It currently includes the following languages:

* English
* Deutsch


== Installation ==
1. Upload the files to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Insert your API-Key and place shortcodes wherever you want.
